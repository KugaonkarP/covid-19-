# covid-19

Android app to track the spread of novel Corona Virus (COVID-19) in near-realtime, thanks to Johns Hopkins University : CSSEGISandData / COVID-19

## Preview
![App Screenshot](screenshots/preview.png)

## Features
- Real-time COVID-19 statistics (Confirmed, Deaths, Recovered)
- Country-wise data breakdown
- Global summary view
- Clean and intuitive UI with Navigation Tab Bar

## Tech Stack
- **Language:** Java
- **API:** Johns Hopkins University CSSE COVID-19 Data
- **UI Components:** Devlight NavigationTabBar
- **Networking:** Retrofit2
- **JSON Parsing:** Gson

## Build the Project with Android Studio
1. Clone the repository
2. Open as existing project in Android Studio
3. Let Gradle sync complete
4. Run on emulator or physical device

## API Reference
Data sourced from [Johns Hopkins CSSE](https://github.com/CSSEGISandData/COVID-19)
